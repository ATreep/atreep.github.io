示例征集计划
2021.1.16 修改并发布 | Treep
关注事项
  • 只有 EasyWeb IDE 2.5.3 及以上版本才可以使用「示例」功能。

    • 背景

    在 EasyWeb IDE 2.5.3 版本中，EasyWeb IDE 的示例功能由内置示例变为在线示例。
    这意味着我们可以随时更新应用内的示例列表，而不必你手动更新应用。
    借此，我们开放了用户投稿渠道。
    我们十分欢迎你将展示你想法或技能的项目打包发送给我们，届时我们会向所有用户公开展示你的项目，共同学习或欣赏你的作品。

    局限类型
    对于你投稿的示项目，我们不会有过于严格的审核标准，不过仍需注意以下几点。
    1.不接受纯广告或无关内容。“征集计划”只为收集对 Web 学习者有益的内容，对于商业化性质较重或无任何价值的投稿项目，我们不会接受（如内含广告链接、友情链接等）。
    2.不接受过于低质的内容。过于简单或无任何技术含量的内容不予展示。
    3.不建议投稿超大型站点。我们建议你打包为 ZIP 文件后的大小不超过 5MB。
    4.不建议投稿 JSP 、 ASP 类型项目，建议 HTML 或 PHP 类型。

    投稿方式
    这里为你提供了三种方式进行投稿。
    1.（推荐）将你的站点打包为 ZIP 文件，上传至快速通道。
    2.将你的站点打包为 ZIP 文件，然后发送至 1261584443@qq.com 邮箱。
    3.将你的站点打包为 ZIP 文件，然后上传至 「EasyWeb IDE 用户交流」群中。
    审核时长
    所有投稿的站点，我们将会按批公开展示。
    由于种种原因，现阶段你的投稿也许会不那么及时的得到展示，多则可能需3至4个周。
    但是，你的投稿长期未得到展示并不意味着审核失败，对于未通过审核的投稿，我们会想尽方式通知你，除非你已失联。
