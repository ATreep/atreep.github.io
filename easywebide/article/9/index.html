关注事项
  • EasyWeb IDE 2.6.5 版本及以上到 2.6.7 版本以下将 EasyApp.js 称做 JSAppActions。

    • EasyApp.js 简介
    EasyApp.js 是一个 JS API，旨在帮助仅熟悉前端的人员轻松打造高度原生化的 Web 转制 Android APP。EasyApp.js 仅适用于 EasyWeb IDE 的「转换为应用」功能。
    无需学习，直接上手！
    我们为 EasyWeb IDE 配备了强大的 快捷插入 功能。因此，你可以在编辑 HTML 或 JS 时点击页面上方功能栏的 [插入] 按钮，一键插入为您预置的 EasyApp.js 语句。
    你也可以点击此处查看我们为你准备的 EasyApp.js（2.6.x） 基础使用演示视频。
    EasyApp.js API 表
    下表适用于 JSAppActions 1.33 。

    File 类 文件系操作

    方法 File.newFile(path)
    描述 返回一个新 File 类实例。
    参数 String path：文件路径。
    返回值 File：File 类实例
    示例 
                                                
//在 SD 卡中新建文件
if(!System.isHaveStoragePermission())
{
    //如果没有存储权限需先申请授权
    System.showRequestStoragePermissionWindow();
}
else
{
    var file=File.newFile("/sdcard/demo.txt");//实例化
    file.createFile(true);
}

    方法 File.createFile(canMakeDirectories)
    描述 创建文件。
    参数 boolean canMakeDirectories：若父目录或更上级目录不存在，是否创建目录后再创建文件。
    返回值 -

    方法 File.createDir(canMakeDirectories)
    描述 创建文件夹。
    参数 boolean canMakeDirectories：若父目录或更上级目录不存在，是否创建目录后再创建文件夹。
    返回值 -

    方法 File.deleteFile(targetFile)
    描述 删除文件（夹）。
    参数 File targetFile：要删除的文件或目录的 File 实例。
    返回值 -
    示例 
                                                
//删除 SD 卡中文件
if(!System.isHaveStoragePermission())
{
    //如果没有存储权限需先申请授权
    System.showRequestStoragePermissionWindow();
}
else
{
    var file=File.newFile("/sdcard/demo.txt");//实例化
    File.deleteFile(file);
}

    方法 File.getName()
    描述 获取 File 实例的目标文件的文件名。
    参数 -
    返回值 String：File 实例的目标文件的文件名。
    示例 
                                                
//获取 SD 卡文件的文件名
if(!System.isHaveStoragePermission())
{
    //如果没有存储权限需先申请授权
    System.showRequestStoragePermissionWindow();
}
else
{
    var file=File.newFile("/sdcard/demo.txt");//实例化
    var name=file.getName();
    //name："demo.txt"
}

    方法 File.getPath()
    描述 获取 File 实例的目标文件的完整路径。
    参数 -
    返回值 String：File 实例的目标文件的完整路径。

    方法 File.getParent()
    描述 获取 File 实例的目标文件的父目录完整路径。
    参数 -
    返回值 String：File 实例的目标文件的父目录完整路径。

    方法 File.getParentFile()
    描述 获取 File 实例的目标文件的父目录 File 实例。
    参数 -
    返回值 String：File 实例的目标文件的父目录 File 实例。
    示例 
                                                
//获取 SD 卡文件的父目录文件名
if(!System.isHaveStoragePermission())
{
    //如果没有存储权限需先申请授权
    System.showRequestStoragePermissionWindow();
}
else
{
    var file=File.newFile("/sdcard/demo.txt");//实例化
    var parentFile=file.getParentFile();
    var parentName=parentFile.getName();
    //parentName："sdcard"
}

    方法 File.getType()
    描述 获取 File 实例的目标文件的拓展名。
    参数 -
    返回值 String：File 实例的目标文件的拓展名（小写字母组成）。若无拓展名则为 null。

    方法 File.copy(file,targetFile)
    描述 复制指定文件至指定目录。请注意，这是一个耗时操作，可能使主线程堵塞。
    参数 File file：待复制文件的 File 实例。
    File targetFile：复制后的 File 实例。
    返回值 -
    示例 
                                                
//复制 SD 卡的 demo.txt 文件至 Download 目录下
if(!System.isHaveStoragePermission())
{
    //如果没有存储权限需先申请授权
    System.showRequestStoragePermissionWindow();
}
else
{
    var file=File.newFile("/sdcard/demo.txt");//实例化
    var targetFile=File.newFile("/sdcard/Download/demo.txt");//实例化
    
    targetFile.getParentFile.createDir(true);//若目标目录不存在则创建

    File.copy(file,targetFile);
}

    方法 File.isFile()
    描述 判断 File 实例是否为文件。
    参数 -
    返回值 boolean：是文件为 true；是文件夹为 false。

    方法 File.isDir()
    描述 判断 File 实例是否为文件夹。
    参数 -
    返回值 boolean：是文件夹为 true；是文件为 false。

    方法 File.exists()
    描述 判断 File 实例的目标文件是否存在。
    参数 -
    返回值 boolean：File 实例的目标文件是否存在。

    方法 File.renameTo(targetFile)
    描述 将 File 实例的目标文件重命名为指定名称。
    参数 File targetFile：重命名后的 File 实例。
    返回值 -
    示例 
                                                
//重命名 SD 卡下的 demo.txt 为 note.txt
if(!System.isHaveStoragePermission())
{
    //如果没有存储权限需先申请授权
    System.showRequestStoragePermissionWindow();
}
else
{
    var file=File.newFile("/sdcard/demo.txt");//实例化
    var targetFile=File.newFile("/sdcard/note.txt");//实例化

    var a=targetFile.exists();

    file.renameTo(targetFile);

    var b=targetFile.exists();
    //a：false；b：true
}

    方法 File.read()
    描述 读取 File 实例的目标文件内容。
    参数 -
    返回值 String：File 实例的目标文件内容。

    方法 File.write(text)
    描述 将指定的内容写入 File 实例的目标文件中。
    参数 String text：写入文本。
    返回值 -

    方法 File.listNames()
    描述 获取 File 实例的目标目录下所有文件名的数组的字符串。
    参数 -
    返回值 String：File 实例的目标目录下所有文件名的数组的字符串。
    示例 
                                                
if(!System.isHaveStoragePermission())
{
    //如果没有存储权限需先申请授权
    System.showRequestStoragePermissionWindow();
}
else
{
    var sdcard=File.newFile(System.getStoragePath());//实例化
    var namesArray=eval(sdcard.listNames());

    var aFileName=namesArray[0];
    var bFileName=namesArray[1];
    var cFileName=namesArray[2];
    //示例
    //aFileName："Android"；
    //bFileName："DCIM"；
    //cFileName："Download"
}

    方法 File.listFiles()
    描述 获取 File 实例的目标目录下所有文件路径的数组的字符串。
    参数 -
    返回值 String：File 实例的目标目录下所有文件路径的数组的字符串。
    示例 
                                                
if(!System.isHaveStoragePermission())
{
    //如果没有存储权限需先申请授权
    System.showRequestStoragePermissionWindow();
}
else
{
    var sdcard=File.newFile(System.getStoragePath());//实例化
    var pathsArray=eval(sdcard.lisFiles());

    var aFilePath=pathsArray[0];
    var bFilePath=pathsArray[1];
    var cFilePath=pathsArray[2];
    //示例
    //aFilePath"/sdcard/Android"；
    //bFilePath"/sdcard/DCIM"；
    //cFilePath"/sdcard/Download"
}

    方法 File.getAppFilesDir()
    描述 获取应用专属目录的 File 实例。
    参数 -
    返回值 String：应用专属目录的 File 实例。例如：getPath()="/data/data/com.venter.app/files"

    方法 File.getAppCacheDir()
    描述 获取应用缓存目录的 File 实例。
    参数 -
    返回值 String：应用缓存目录的 File 实例。例如：getPath()="/data/data/com.venter.app/cache"

    方法 File.copyFileFromSite(sitePath,targetFile)
    描述 将站点目录下的指定文件复制到 targetFile 所对应的位置。
    参数 String sitePath：相对于站点根目录的文件相对路径。
    File targetFile：复制到设备上的目标位置的 File 实例。
    返回值 -
    示例 
                                                
if(!System.isHaveStoragePermission())
{
    //如果没有存储权限需先申请授权
    System.showRequestStoragePermissionWindow();
}
else
{
    var sitePath="doc/myfile";//站点目录下的 doc 文件夹中的 myfile 文件
    var target=File.newFile(System.getStoragePath()+"/myfile");//实例化
    File.copyFileFromSite(sitePath,target);
    
    target.exists();//返回 true
    
}


    UI 类 用户界面系操作

    方法 UI.showToast(msg,isLongDuration)
    描述 显示一条 Toast 弹出消息。
    参数 String msg：显示信息；
    boolean isLongDuration：是否使 Toast 持续显示较长时间。
    返回值 -

    方法 UI.setTitle(text)
    描述 设置标题栏文本内容。
    参数 String text：标题。
    返回值 -

    方法 UI.getTitle()
    描述 获取标题栏文本内容。
    参数 -
    返回值 String：当前标题栏内容。

    方法 UI.setTitleVisible(isVisible)
    描述 设置标题栏是否显示。
    参数 boolean isVisible：是否显示标题栏。
    返回值 -

    方法 UI.getTitleVisible()
    描述 获取标题栏是否显示。
    参数 -
    返回值 boolean：标题栏是否显示。

    方法 UI.getStatueBarHeight()
    描述 获取状态栏高度。
    参数 -
    返回值 int：状态栏高度值（单位：PX）。
    示例 
                                                
setTitleVisible(false);//使标题栏隐藏，网页填充至整个页面（包括状态栏）
//为使网页主体内容不与状态栏重叠，需设置间距
document.body.style.padding=UI.getStatueBarHeight()+"px";

    返回值 -

    返回值 -

    参数 -
    返回值 -

    方法 UI.allowSkip( isAllowSkip )

    System 类 系统交互系操作

    方法 System.isHaveStoragePermission()
    描述 判断是否拥有存储权限，即 WRITE_EXTERNAL_STORAGE 权限。
    参数 -
    返回值 boolean：是否拥有存储权限。

    方法 System.showRequestStoragePermissionWindow()
    描述 弹出申请存储权限窗口（若已授权可能不会弹出）。
    参数 -
    返回值 -

    方法 System.getStoragePath()
    描述 获取内部存储（SD 卡）路径。
    参数 -
    返回值 String：内部存储路径，一般为”/storage/emulated/0“。

    方法 System.startApp(packageName)
    描述 启动指定应用，若应用不存在会引发异常。
    参数 String packageName：应用包名。如：com.venter.app。
    返回值 boolean: 是否打开成功。

    方法 System.exitApp()
    描述 关闭此应用。
    参数 -
    返回值 -

    方法 System.clearData()
    描述 清除所有浏览器数据。
    执行此操作后，应用产生的所有 Cookies 及加载缓存等都会被清除。
    参数 -
    返回值 -
