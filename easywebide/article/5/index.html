「快速开始」页面自定义方法表及示例
2020.4.4修改后发布 | Treep
关注事项
  • 只有 EasyWeb 2.2.3 及以上版本才支持自定义「快速开始」页面。

    • 开始
    在 EasyWeb 2.2.3 版本中，你可以编写一个属于自己的精美主页并将它应用在「快速开始」页面。为了保证你的新主页可以与EasyWeb灵活交互（例如在你的自定义主页上添加一个“创建新项目”按钮），你有必要参考 EasyWeb 为您提供的方法表（API）。当然 EasyWeb 为您准备了一些示例代码，帮助你快速理解一些API的作用。
    你可以下载EasyWeb默认「快速开始」页面进行参考。点此下载。
    这里有一个自定义「快速开始」页面的演示视频，点此查看。

    表
    EasyWeb.createWebsite()
    弹出创建站点对话框。
    无返回值。
    无参数。
    EasyWeb.createTempWebsite()
    创建临时站点。
    无返回值。
    无参数。
    EasyWeb.createFileType()
    弹出新建文件类型对话框。
    无返回值。
    无参数。
    EasyWeb.openURL(String url)
    使用第三方浏览器打开指定url。
    无返回值。
    String url:指定打开的链接（如："http://www.example.com/"）。
    EasyWeb.getTheme()
    获取当前应用主题值。
    返回 int 类型主题值。[-1:默认,0:经典,1:夜间,2:清新,3:淡雅]
    无参数。
    EasyWeb.isLogined()
    获取是否创建过本地账户。
    返回 boolean 类型值。[true:已创建本地账户,false:未创建本地账户]
    无参数。
    EasyWeb.showLogin()
    弹出创建本地账户对话框。
    无返回值。
    无参数。
    EasyWeb.hasRecover()
    EasyWeb为您准备了“恢复模式”示例，请转至下一部分。
    获取是否存在备份。
    返回 boolean 类型值。[true:存在上一次的备份。,false:没有上一次的备份]
    无参数。
    EasyWeb.delRecover()
    若存在备份则删除备份。
    无返回值。
    无参数。
    EasyWeb.getRecoverTime()
    若存在备份则获取备份创建时间。
    返回 String 类型值。
    无参数。
    EasyWeb.recover()
    若存在备份则立即恢复工作区。若无备份则不要调用此方法。
    无返回值。
    无参数。
    EasyWeb.getRecentProjects()
    EasyWeb为您准备了“罗列最近项目”示例，请转至下一部分。
    获取最近打开的项目（最多最近5个项目）。
    返回 String 类型值。
    无参数。
    EasyWeb.openRecentWebsite(int index)
    打开指定的最近项目。
    无返回值。
    int index:项目索引。使用 EasyWeb.getRecentProjects() 方法获取到一个数组后，此参数则对应数组索引。
    示例
    “恢复模式示例”
    index.html: 
                                
<html>
    ······
    <div id="recover">
        <span>来自上一次的备份</span>
        <br>
        <button type="button" id="recover_btn" onclick="javascript:EasyWeb.recover()"></button>
    </div>
    ······
</html>
    main.js: 
            
var recover=document.getElementById("recover");//获取显示恢复区域的div。
var recover_btn=document.getElementById("recover_btn");
if(EasyWeb.hasRecover)
{
    //存在备份
    var time=EasyWeb.getRecoverTime();//获取备份创建时间。
    recover_btn.innerHTML=time+"的备份";
}
else
{
    //不存在备份，隐藏恢复div。
    recover.style.display="none";
}
